# first-repository

학습과 연습을 위한 개인 저장소입니다. 새로운 도구와 워크플로를 실험하고 기록하는 공간으로 사용합니다.

## 목적

- Git과 GitHub 협업 흐름(브랜치, 커밋, 풀 리퀘스트) 연습
- 다양한 도구와 기술을 실습하며 메모와 예제를 남기기

## 시작하기

저장소를 로컬에 복제합니다.

```bash
git clone https://github.com/jiyongham/first-repository.git
cd first-repository
```

## 기여 방법

작은 변경이라도 다음 흐름을 따릅니다.

1. 작업용 브랜치를 만듭니다. (`git switch -c <branch-name>`)
2. 변경 사항을 커밋합니다.
3. 브랜치를 푸시하고 풀 리퀘스트를 엽니다.

## 라이선스

별도로 명시하지 않은 한 개인 학습용으로 자유롭게 참고할 수 있습니다.
